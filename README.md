Fitbit provider fot authentication and data collection.

Developers can register applications to use the Fitbit API at <https://dev.fitbit.com/apps>.

## Installation

To install, use composer:

```bash
$ composer require jbtcd/fitbit
```

## Testing

``` bash
$ composer run tests
```
